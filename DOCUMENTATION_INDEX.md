## Getting Help

### Documentation Issues

If you find issues with documentation:

1. **Check if it's outdated**: Compare with current code
2. **Search existing issues**: [GitHub Issues](https://github.com/luigi043/New-Insurex/issues)
3. **Create new issue**: Use "documentation" label
4. **Submit PR**: Fix it yourself and contribute

### Technical Support

For technical issues:

1. **Check troubleshooting sections** in relevant docs
2. **Search GitHub Issues** for similar problems
3. **Review logs** in `InsureX.API/logs/`
4. **Create detailed issue** with:
   - Error messages
   - Steps to reproduce
   - Environment details
   - Logs and screenshots

### Contributing

Want to improve documentation?

1. Fork the repository
2. Create branch: `docs/description`
3. Make changes
4. Submit pull request
5. Reference issue if applicable
